# Overview

page-agent is a purely web-based GUI Agent. Gives your website an AI operator in simple steps.

## What is page-agent?

page-agent is an embedded GUI Agent. Unlike traditional browser automation tools, page-agent is built for web developers and web applications first. Integrate it into your site to let users interact with pages through natural language.

## Core Features

- **Smart DOM Analysis** — DOM-based analysis with high-intensity dehydration. No visual recognition needed. Pure text for fast and precise operations.
- **Secure & Controllable** — Supports operation allowlists, data masking protection. Inject custom knowledge to make AI work by your rules.
- **Zero Backend** — CDN or NPM import with custom LLM endpoints.
- **Accessible Intelligence** — Provides natural language interface for complex B2B systems and admin panels. Makes software easy for everyone.

## vs. browser-use

| | page-agent | browser-use |
|---|---|---|
| Deployment | Embedded component | External tool |
| Scope | Current page | Entire browser |
| Target Users | Web developers | Scraper/Agent developers |
| Use Case | UX enhancement | Automation tasks |

## Use Cases

1. **Connect Support Bots:** Turn your support assistant into a full agent. Customer service bots no longer just say "Please click the settings button then click..."—they operate for users directly.
2. **Modernize Legacy Apps:** One line of code transforms old apps into agents. Product experts help users navigate complex B2B software. Reduce support costs and improve satisfaction.
3. **Interactive Training:** Demonstrate workflows in real-time. Let AI show the complete process of "how to submit an expense report."
4. **Accessibility:** Provide natural language interaction for visually impaired and elderly users. Connect screen readers or voice assistants to make software accessible to everyone.
